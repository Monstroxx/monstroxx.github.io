Welcome to your new blog powered by the Moonwalk theme! This is a fast and minimalistic Jekyll theme with clean dark mode support.

## Features

- Light & dark mode with theme switcher
- Fast performance
- Responsive design
- SEO optimized

You can start writing your blog posts in the `_posts` directory. Just create a new markdown file with the format `YYYY-MM-DD-post-title.md`.
