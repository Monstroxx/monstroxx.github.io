## Local Development

1. Install dependencies:
```bash
bundle install
```

2. Start the development server:
```bash
bundle exec jekyll serve
```

3. Open your browser at `http://localhost:4000`

## Deployment

This site is configured for GitHub Pages. Simply push to the repository and GitHub Pages will automatically build and deploy your site.

## Customization

- Edit `_config.yml` for site settings
- Modify `_data/home.yml` for menu and project entries
- Add blog posts in `_posts/` directory
- Customize colors in SCSS files
